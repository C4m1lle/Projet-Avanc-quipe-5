Initialise le gestionnaire de signal pour l'algorithme de bruteforce.

Cette fonction configure le handler pour le signal SIGINT (Ctrl+C), puis lance l'algorithme de bruteforce, soit en version classique, soit avec la demi-matrice selon le paramètre fourni.

Paramètres
tourTournée initiale.
distFonction de calcul de distance.
bestTableau stockant la meilleure permutation.
distminPointeur vers la distance minimale trouvée.
bffBooléen indiquant le choix de la version du bruteforce (0 = classique, 1 = demi-matrice).
